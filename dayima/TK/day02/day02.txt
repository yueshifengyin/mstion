day02笔记:
布局 Layout
  tkinter布局的种类:
    pack 打包布局
    grid 栅格布局
    place 放置布局(几乎不用)

打包布局 pack
  此种布局类似向行李箱里摆放东西

pack的常用属性:
   side  停靠窗口的位置
   ipadx  横向内边距
   ipady  纵向内边距
   padx   横向外边距
   pady   纵向外边距
   fill   填充方向
   expand 是否填充整个空白区域(默认为0)

side 属性:
  作用：
    用来设置打包放置边
  值:
    tkinter.
      左LEFT, 右 RIGHT, 上(TOP), 下(BOTTOM)

fill属性:
  作用：
     用于在某个方向上填充空白区域
    值:
       tkinter.
         None (默认)
         X    (水平方向)
         Y    (竖直方向)
         BOTH (水平竖直两个方向)

expand 属性
  作用：
    填充整个空白区域
  值:
    0 不扩展(默认)
    1 扩展
  说明：
    当expand值为1时，side 属性设置无效


ipadx/ipady
  作用：
    设置内边距的值,默认单位为像素
  值:
    0~n的整数值

padx/pady
  作用：
    设置外边距的值，默认单位为像素
  值:
    0~n的整数(默认为0)

什么是内边距和外边距?


grid 栅格布局:
  grid 常用属性:
    row    行
    colume 列
    ipadx  横向内边距
    ipady  纵向内边距
    padx   横向外边距
    pady   纵向外边距
    rowspan 跨越行数
    columnspan 跨越列数
    stick      粘滞方向(N, E, S, W)

Frame 控件
  作用：
    形成一个单独的可用来布局的区域,相当于控件的容器
  常用属性:
    bg/background   背景色
    height          高
    width           宽
    borderwidth     边框宽度


窗口部件widget的config方法:
  作用：
    用于修改widget对象的属性
  语法:
    xxx.config(属性1=值1, 属性2=值2, ...)

示例：
   label = tkinter.Label(root, text="hello")
   lable.config(text='world')  # 将label的text属性改为'world' 字符串
